# Train Watcher
## What is this?
[Vivi](https://vivi.lv), the Latvian passenger train company, has a [Web App](https://trainmap.vivi.lv/) for tracking the train locations.  
I figured, "hey, I don't like it, so let me make something simpler for tracking it". So here we are!  

... and also that they send ≈1MB per second ...
